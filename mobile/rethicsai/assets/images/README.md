## Usage Guidelines

1. All images should be optimized for mobile devices
2. Use PNG format with transparency where applicable
3. Maintain consistent color scheme (primary: #1976D2, secondary: #388E3C)
4. Images should be culturally appropriate for African audiences
5. Include diverse representation where applicable
